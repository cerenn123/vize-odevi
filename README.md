# vize-odevi
Bu kod, NCBI GEO’dan  gen ekspresyon verilerini alarak örneklerin gen ifadelerini karşılaştırmamızı sağlar. Veriyi normalize ettikten sonra K-Means algoritmasıyla örnekleri kümelere ayırır ve PCA ile 2 boyuta indirerek görselleştirir. Böylece benzer gen profillerine sahip örnekleri hızlıca tespit edip, biyolojik ilişkileri analiz edebiliriz.
